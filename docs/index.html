핵심 기능

🏪 다중 고객 사업장

한 시스템에서 여러 소상공인 사업장 독립 운영. 완전한 tenant 격리.

🧠 RAG 지식 베이스

사업장 자료 업로드 → 자동 청킹/임베딩 → AI 직원이 즉시 활용.

🤖 AI 직원 페르소나

톤, 어휘, 금지어, handoff 규칙을 사업자별로 정의.

📅 주간 자동 일정

월~일 자동 업무 생성, 검수, 발행.

📱 멀티 채널 발행

Instagram, Threads, Mastodon, Naver, Kakao 공식 API.

🔁 자동화 규칙

FAQ 자동응대, 주간 리포트, 콘텐츠 자동 생성.

📊 결과 리포트

일간/주간 자동 리포트 + 개선 제안.

🔌 Hermes 연동

외부 Hermes Agent와 자동화 규칙 결과를 주고받을 수 있음.
